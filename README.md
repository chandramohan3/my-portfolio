# Chandramohan S — Portfolio (Final)

This is a Tailwind CLI based portfolio project packaged for easy use.

## Quick start

1. Install dependencies:
```bash
npm install
```

2. Start Tailwind in watch mode (generates dist/output.css):
```bash
npm run watch
```

3. In a new terminal, serve the project locally (optional):
```bash
npx serve -s . -l 5000
```

4. Open `http://localhost:5000`

Replace `src/assets/profile.jpg` with your preferred photo if needed. `resume.pdf` is included.
